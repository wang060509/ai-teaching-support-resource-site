# Project Overview

## Summary

AI Teaching Support Knowledge Base is a teaching-support resource site focused on turning AI learning resources into accessible materials for higher education.

The site documents beginner-friendly AI literacy topics, teaching use cases, and reusable learning materials without publishing private or raw project materials.

## Project Goals

| Goal | Description |
| --- | --- |
| Organize AI learning topics | Group concepts into clear areas that can support future teaching. |
| Translate for beginners | Convert complex AI ideas into concise, approachable explanations. |
| Prepare reusable assets | Shape guides, module maps, and workshop materials that can be adapted later. |
| Support responsible AI practice | Emphasize verification, privacy awareness, and human review. |

## Scope

This version includes page structures and placeholder guide pages. Detailed content will be added only after manual review.

::: warning Not an official site
This website is a personal learning and teaching-support site. It should not be presented as the official website of RAISE Hub or any university office.
:::

## Content Boundaries

| Excluded material | Reason |
| --- | --- |
| Raw sources | May contain unreviewed or private content |
| Professor emails | Private communication |
| Approval PDFs | Administrative documentation |
| Budget materials | Internal operational information |
| Internal Drive instructions | Restricted workflow information |
| Private RAISE Hub administrative files | Not appropriate for a public teaching-resource site |

## Current Guide Areas

- AI evaluation for beginners
- AI capabilities and limitations
- Beginner vibe coding
- Codex beginner workflow
- Workshop module planning

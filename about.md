# About

## Project Identity

AI Teaching Support Knowledge Base is a personal learning and teaching-support site. It presents beginner-friendly AI literacy resources, guide drafts, and reusable workshop preparation materials.

It is not an official RAISE Hub website.

## Purpose

| Purpose | Description |
| --- | --- |
| Teaching documentation | Organize AI topics into beginner-friendly summaries and reusable structures. |
| Workshop support | Prepare guide and module pages that can support classroom or workshop use. |
| Review workflow | Keep drafted materials easy to check, adapt, and revise for specific teaching contexts. |

## What This Site Includes

- Project overview and workflow notes
- Drafted beginner-friendly guides and module planning materials
- Concise descriptions of intended teaching use
- Review guidance for adapting materials to classroom or workshop use

## What This Site Excludes

::: warning Private materials are not included
This site does not copy raw sources, professor emails, approval PDFs, budget files, internal Drive instructions, or private RAISE Hub administrative documents.
:::

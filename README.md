# AI Teaching Support Resource Site

A VitePress-based teaching support site for beginner-friendly AI literacy materials, responsible AI use, and reusable workshop preparation.

Live site: https://wang060509.github.io/ai-teaching-support-resource-site/

## Project Purpose

This site organizes AI teaching-support materials into clear, beginner-friendly guides and module planning pages.

It is designed for students, faculty, workshop participants, and teaching support staff who need practical ways to understand, evaluate, and use AI tools responsibly.

This is a personal learning and teaching-support resource site. It is not an official institutional website.

## What This Site Includes

- AI capabilities and limitations guide
- AI output evaluation beginner guide
- AI-assisted project workflow guide
- Codex-style beginner workflow guide
- Workshop module map
- Teaching material workflow notes
- Clear privacy and content-boundary notes

## Guides & Modules

| Page | Teaching Focus |
|---|---|
| AI Evaluation Beginner Guide | Helps beginners review AI-generated output before trusting or using it |
| AI Capabilities and Limitations | Explains what AI tools can and cannot reliably do |
| AI-Assisted Project Workflow | Introduces small-scope AI-assisted project work for beginners |
| Codex Beginner Workflow | Explains Codex-style AI workflow habits in a tool-agnostic way |
| Workshop Module Map | Connects guide topics into reusable teaching or workshop modules |

## Teaching Approach

The site emphasizes:

- calibrated trust rather than hype or fear
- beginner-friendly explanations
- human review and verification
- privacy-aware AI use
- small, reviewable learning activities
- reusable teaching materials

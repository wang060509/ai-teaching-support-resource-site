## Teaching Resource Focus

This site organizes AI teaching-support materials for beginner-friendly learning, classroom use, and workshop planning. It is a personal learning and teaching-support site, not an official institutional website.

| Focus area | Teaching use | Current status |
| --- | --- | --- |
| AI evaluation | Helps beginners judge AI outputs with clearer criteria | Guide drafted |
| Capabilities and limitations | Explains what AI tools can and cannot reliably do | Guide drafted |
| AI-assisted workflow | Introduces exploratory project workflows for beginners | Guide drafted |
| Codex workflow | Documents a beginner-friendly assistant-assisted development process | Guide drafted |
| Workshop planning | Maps topics into reusable learning modules | Module map drafted |

::: info Content boundary
This site uses concise summaries and placeholders. It does not copy raw sources, professor emails, approval PDFs, budget materials, internal Drive instructions, or private administrative documents.
:::
